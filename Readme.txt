       x--------------------SNAKE GAME PROJECT------------------x

    This project is a simple implementation of the classic Snake game using HTML, CSS, and JavaScript.

x-------------------------Installation-------------------------x
            To run the game locally, you'll need to:

            Clone this repository to your local machine.
            Open the index.html file in your browser.
X--------------------------------Usage---------------------------X
            Use the arrow keys on your keyboard to control the direction of the snake.
             The objective is to eat as many apples as possible without hitting the walls or the snake's own tail.
              The game ends when the snake hits a wall or its own tail.

X--------------------------Development--------------------------X
            This project was developed using HTML, CSS, and JavaScript. The game logic is implemented in script.js.
